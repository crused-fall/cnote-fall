> [!猜想]
> $a_1=a>0$，序列$a_n$满足递归关系$$a_{n+1}=a_n+\frac{n^{2\beta}}{a_1+\cdots+a_n}$$那么$$\lim_{n\to \infty} \frac{a_n}{n^{\beta}}=\sqrt{\frac{\beta+1}{\beta}}$$

### 1. $\beta=\frac{1}{2}$特例的解答

也就是说序列满足的递推关系为:$$a_{n+1}=a_n+\frac{n}{S_n}$$
#### 1.1 假设极限存在，极限是多少?

因为递推关系给出了差分的表达，因此可以考虑利用Stolz的工具。其中的技巧在于$S_n = a_1+\cdots+a_n$其差分满足$S_{n+1}-S_n = a_{n+1}$。通过多次差分，结合Stolz可以得到关于$\lim_{n\to \infty} \frac{a_n}{\sqrt{n}}$的一个方程，从而得到极限是$\sqrt{3}$。

具体来说$$\frac{a_n}{\sqrt{n}}\sim \frac{a_{n+1}-a_n}{\sqrt{n+1}-\sqrt{n}}\sim \frac{2n^{3/2}}{S_n}\sim\frac{3\sqrt{n+1}}{a_{n+1}}$$如果极限是存在的，那么我们就能得到一个方程，从而得到极限为$\sqrt{3}$

然而这一切都基于这个极限是存在的基础上，因此事先应该证明极限的存在性。极限的存在性才是这个问题解决的关键。

#### 1.2 极限存在性的证明

#### 1.3 单调性？

是否可以证明序列$\frac{a_n}{\sqrt{n}}$的某种单调性?

```wolfram
Clear[a, b, s, n, m, data]
m = 10000;
(*初值设定为随机正实数*)
b = RandomReal[{0.1, 50}]
(*得到关于序列迭代的数据*)
data = RecurrenceTable[{a[n + 1] == a[n] + n/s[n], 
s[n + 1] == s[n] + a[n + 1], a[1] == b, s[1] == a[1]}, {a, s}, {n, 1, m}][[All, 1]];
(*生成图像*)
ListPlot[Table[data[[k]]/Sqrt[k], {k, 1, m}]]
```

随机数的初值告诉我们，似乎这个序列当$n$足够大的时候，序列是单调增加到$\sqrt{3}$的。也就是说我们猜测序列：

> [!猜想1.3]
> 当$n$足够大的时候序列$b_n:=\frac{a_n}{\sqrt{n}}$:
> 1. $b_n$单调增加
> 2. $b_n$有上界

$$\begin{aligned}\frac{a_{n+1}}{\sqrt{n+1}}-\frac{a_n}{\sqrt{n}}&=\frac{\sqrt{n}-\sqrt{n+1}}{\sqrt{n+1}\sqrt{n}}a_n+\frac{n}{\sqrt{n+1}}\frac{1}{S_n}\end{aligned}$$那么此序列想要单调增加，就必须在$n$足够大的时候满足$$\frac{n^{3/2}}{\sqrt{n+1}-\sqrt{n}}>a_nS_n$$这并不容易证明，因为如果单纯假设当$n$足够大的时候$a_n<\sqrt{3n}$,无论我们如何估计$S_n$的上界，都无法满足这个不等式。

#### 1.4 证明上极限等于下极限？

如果假设$a = \liminf_{n\to \infty}\frac{x_n}{\sqrt{n}},b=\limsup_{n\to \infty}\frac{x_n}{\sqrt{n}}$,那么我们的目标无非就是证明$a=b=\sqrt{3}$,当然这里是分析问题，所以实际上我们要证明的是$$\sqrt{3}\leq a\leq b\leq \sqrt{3}$$首先还是从Stolz-Cesaro入手。虽然我们不知道极限是否存在，但是我们可以利用广义的Stolz-Cesaro的结果。（参考：[[Cesaro-Stolz定理]]）

首先，我们排除一开始1.1当中的$$\frac{a_n}{\sqrt{n}}\sim \frac{a_{n+1}-a_n}{\sqrt{n+1}-\sqrt{n}}\sim \frac{2n^{3/2}}{S_n}\sim\frac{3\sqrt{n+1}}{a_{n+1}}$$这一套流的可能性，上面的这一套并不能向我们提供关于$a,b$的有用的不等式。因为参考[[关于上下极限的几个错误命题]]当中的“错误命题1”，告诉我们，在极限存在性尚未明确的情况下，我们不能确定$\frac{a_n}{\sqrt{n}}$的上(下)极限的倒数与$\frac{\sqrt{n}}{a_n}$的上(下)极限之间的大小关系。我们最多能从[[关于上下极限的几个错误命题]]当中的“正确命题3”当中得到信息$$ab\leq 3,ab\geq 3\iff ab=3$$但是这样并不足以让我们知道$a=b=\sqrt{3}$。

不过由一开始的这个结果我们知道$$2\limsup_{n\to \infty}\frac{n^{3/2}}{S_n}\geq b\geq a\geq 2\liminf_{n\to \infty}\frac{n^{3/2}}{S_n}$$虽然我们知道继续做差分下去是徒劳的，不过我们想到[[关于上下极限的几个错误命题]]当中的“正确命题2”，我们不妨考虑其平方的关系,因为$$4\limsup_{n\to \infty}\frac{n^{3}}{S_n^2}\geq b^2\geq a^2\geq 4\liminf_{n\to \infty}\frac{n^{3}}{S_n^2}$$所以我们现在的兴趣在于序列$\frac{n^3}{S_n^2}$,不过为了广义Stolz定理需要的那个分母的单调性条件的满足，这里我们利用[[关于上下极限的几个错误命题]]当中的"正确命题3"的结果，调换一下序列的上下关系。比如我们考虑用$\left(\limsup_{n\to \infty}\frac{S_n^2}{n^{3}}\right)^{-1}$替换$\liminf_{n\to \infty}\frac{n^{3}}{S_n^2}$,以及$\left(\liminf_{n\to \infty}\frac{S_n^2}{n^{3}}\right)^{-1}$替换$\limsup_{n\to \infty}\frac{n^{3}}{S_n^2}$。
$$\begin{aligned}\liminf_{n\to \infty}\frac{S_n^2}{n^3}&\geq \liminf_{n\to \infty}\frac{S_{n+1}^2-S_n^2}{3n^2}\\&\geq \liminf_{n\to \infty}\frac{S_{n+1}^2-2S_n^2+S_{n-1}^2}{6n}\\&=\liminf_{n\to \infty}\frac{2n+a_{n+1}^2+a_n^2}{6n}\\&=\frac{a^2+1}{3}\end{aligned}$$
* 此处的第一个处理：我们从原本序列的递推关系当中得到了$$S_{n+1}^2-2S_n^2+S_{n-1}^2=2n+a_{n+1}^2+a_n^2$$
于是上面的推导告诉我们$$\liminf_{n\to \infty}\frac{S_n^2}{n^3}=\left(\limsup_{n\to \infty}\frac{n^{3}}{S_n^2}\right)^{-1}\geq \frac{a^2+1}{3}$$也就是说$$b^2\leq \frac{12}{a^2+1}$$同样的方式还可以得到
$$a^2\geq \frac{12}{b^2+1}$$
两个式子结合起来,再结合$ab=3,0<a\leq b$,我们可以解出这个二元不等式组的唯一解$$a=b=\sqrt{3}$$于是我们便证明了上下极限相等，从而证明了极限的存在性。

### 2. 特例$\beta=1$的解答

这种情况下的递推关系为$$a_{n+1}=a_n+\frac{n^{2}}{S_n}$$我们需要证明$$\lim_{n\to \infty} \frac{a_n}{n}=\sqrt{2}$$
首先在极限存在的情况下我们很容易通过Stolz-Cesaro定理验证极限必定是$\sqrt{2}$。但是，通过第一节我们已经知道，该问题的真正困难在于极限的存在性，并且验证该类型的序列的单调性非常困难，于是我们打算参照第一节的方式，令$$a=\liminf_{n\to \infty} \frac{a_n}{n},b=\limsup_{n\to \infty} \frac{a_n}{n}$$我们的任务是证明$$\sqrt{2}\leq a\leq b\leq \sqrt{2}$$首先在关于对$\frac{a_n}{n}$使用广义的Stolz-Cesaro定理，我们会得到$$\begin{aligned}a&\geq \liminf_{n\to \infty} a_{n+1}-a_n\\&\geq \liminf_{n\to \infty} \frac{n^{2}}{S_n}\\&\geq \liminf_{n\to \infty}\frac{2n+1}{a_{n+1}}\\&= 2\left(\limsup_{n\to \infty}\frac{a_n}{n}\right)^{-1}=\frac{2}{b}\end{aligned}$$同样的方式如果一开始是$b$的话，我们能得到相反的不等式，于是$$ab=2$$然后我们考虑上述过程中的一个序列$\frac{n^{2}}{S_n}$,我们知道$$\limsup_{n\to \infty}\frac{n^{4}}{S_n^2}\geq b^2\geq a^2\geq \liminf_{n\to \infty}\frac{n^{4}}{S_n^2}$$现在我们关心序列$\frac{n^{4}}{S_n^2}$，并想要尝试对其进行广义Stolz-Cesaro定理的差分处理。不过为了广义Stolz定理需要的那个分母的单调性条件的满足，这里我们利用[[关于上下极限的几个错误命题]]当中的"正确命题3"的结果，调换一下序列的上下关系。比如我们考虑用$\left(\limsup_{n\to \infty}\frac{S_n^2}{n^{4}}\right)^{-1}$替换$\liminf_{n\to \infty}\frac{n^{4}}{S_n^2}$,以及$\left(\liminf_{n\to \infty}\frac{S_n^2}{n^{4}}\right)^{-1}$替换$\limsup_{n\to \infty}\frac{n^{4}}{S_n^2}$。$$\begin{aligned}\liminf_{n\to \infty}\frac{S_n^2}{n^{4}} &\geq 4\liminf_{n\to \infty}\frac{S_{n+1}^2-S_n^2}{n^3}\\&\geq 12\liminf_{n\to \infty}\frac{S_{n+1}^2-2S_n^2+S_{n-1}^2}{n^2} \\&\geq 12\liminf_{n\to \infty} \frac{a_{n+1}^2+a_n^2+2n^2}{n^2} \\&= \frac{a^2+1}{6}\end{aligned}$$
* 此处第一点，我们利用了从原本的递推关系当中得到的递推关系$$S_{n+1}^2-2S_n^2+S_{n-1}^2=2n^2+a_{n+1}^2+a_n^2$$
于是上面的推导告诉我们$$\liminf_{n\to \infty}\frac{S_n^2}{n^4}=\left(\limsup_{n\to \infty}\frac{n^{4}}{S_n^2}\right)^{-1}\geq \frac{a^2+1}{6}$$也就是说$$b^2\leq \frac{6}{a^2+1}$$
同样的方式，还可以得到$$a^2\leq \frac{6}{b^2+1}$$结合其他关系$ab=2,0<a\leq b$等，我们可以解的唯一的结果$$a=b=\sqrt{2}$$
### 3. 可以把以上的方法推广到任意$\beta>0$吗？

$$a=\liminf_{n\to \infty} \frac{a_n}{n^{\beta}},b=\limsup_{n\to \infty} \frac{a_n}{n^{\beta}}$$令$r_{\beta}:=\frac{\beta+1}{\beta}$
$$\sqrt{r_{\beta}}\leq a\leq b\leq \sqrt{r_{\beta}}$$首先我们可以得到一个相较于最终结果弱一些的结果:

> [!命题3.1]
> 存在两个正实数$c_1,c_2$使得$$0<c_1<a\leq b<c_2<+\infty$$

然后和上面1，2节的做法一致，首先尝试广义Stolz-Cesaro定理对序列$\frac{a_n}{n^{\beta}}$的处理:$$\begin{aligned}a&\geq \liminf_{n\to \infty} \frac{a_{n+1}-a_n}{\beta n^{\beta-1}}\\&=\frac{1}{\beta }\liminf_{n\to \infty} \frac{n^{\beta+1}}{S_n}\\&\geq \frac{\beta+1}{\beta}\liminf_{n\to \infty}\frac{n^{\beta}}{a_{n}}\\&= r_{\beta}\left(\limsup_{n\to \infty}\frac{a_n}{n^{\beta}}\right)^{-1}=r_{\beta}\frac{1}{b}\end{aligned}$$同样我们也可以证明相反的不等式，于是我们得到$$ab=r_{\beta}$$然后我们关注上述过程中的$\frac{n^{\beta+1}}{S_n}$这个序列，重复1,2节当中类似的过程，我们注意到$$\liminf_{n\to \infty} \frac{n^{2\beta+2}}{S_n^2} \leq a^2\leq b^2\leq \limsup_{n\to \infty}\frac{n^{2\beta+2}}{S_n^2}$$我们现在尝试对序列$\frac{n^{2\beta+2}}{S_n^2}$使用广义的Stolz-Cesaro定理,不过为了广义Stolz定理需要的那个分母的单调性条件的满足，这里我们利用[[关于上下极限的几个错误命题]]当中的"正确命题3"的结果，调换一下序列的上下关系。比如我们考虑用$\left(\limsup_{n\to \infty}\frac{S_n^2}{n^{2\beta+2}}\right)^{-1}$替换$\liminf_{n\to \infty}\frac{n^{2\beta+2}}{S_n^2}$,以及$\left(\liminf_{n\to \infty}\frac{S_n^2}{n^{2\beta+2}}\right)^{-1}$替换$\limsup_{n\to \infty}\frac{n^{2\beta+2}}{S_n^2}$。$$\begin{aligned}\liminf_{n\to \infty}\frac{S_n^2}{n^{2\beta+2}} &\geq (2\beta+2)\liminf_{n\to \infty}\frac{S_{n+1}^2-S_n^2}{n^{2\beta+1}}\\&\geq (2\beta+2)(2\beta+1)\liminf_{n\to \infty}\frac{S_{n+1}^2-2S_n^2+S_{n-1}^2}{n^{2\beta}} \\&\geq (2\beta+2)(2\beta+1)\liminf_{n\to \infty} \frac{a_{n+1}^2+a_n^2+2n^{2\beta}}{n^{2\beta}} \\&= \frac{(\beta+1)(2\beta+1)}{a^2+1}\end{aligned}$$于是我们得到不等式$$b^2\leq \frac{(\beta+1)(2\beta+1)}{a^2+1}$$同样的计算，我们还能得到$$a^2\geq \frac{(\beta+1)(2\beta+1)}{b^2+1}$$结合$ab=r_{\beta}$,以及命题3.1当中的$0<c_1<a\leq b<c_2<+\infty$，我们可以解得唯一的一组解$$a=b=\sqrt{r_{\beta}}$$于是极限存在，并且等于$\sqrt{\frac{\beta+1}{\beta}}$。
